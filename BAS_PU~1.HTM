ⓘ How to use this simulator ▾
  1. Set pond dimensions — pick Aggregate (type totals) or Customize (paint individual ponds on the aerial). Press Apply & Reset to commit.
  2. Configure Sea Pumps — count (±), avg flow per pump, Pump A duty hours. "Per Cycle" pill shows daily delivery.
  3. Set Feed & Water Demand — kg / 2,500 m² + m³ / kg standard. Compliance gauge shows if pumps can meet demand.
  4. Read the health box — GREEN balanced · PURPLE OK + advisories · AMBER review · RED failure predicted.
  5. Run — press ▶ Play; slider/presets 0.5× → 100× to fast-forward.
  6. Reset — ↻ Reset = pumps/feed/duty/speed to defaults · Set Default & Reset = pond dimensions to factory.
  7. ☰ Scenarios (top-right) — save/load/compare/print named configurations.
    • Type a name, press 💾 Save to store current config (duplicates blocked).
    • Click a name to rename inline. Load/Update per row.
    • Tick rows then use 🗑 Delete · ⇆ Compare (1 = vs live, 2–4 = against each other) · 🖨 Print (1 A4 / scenario).
    • Compare highlights diffs in yellow + shows saved health badge (✓ ℹ ⚠ ✕) per scenario.

💡 Purple "TP Too Small" still runs but never hits 100% compliance — enlarge TP1/TP2 or reduce feed.

📋 Save before tweaking; Load to revert; ⇆ Compare to evaluate "what if".
